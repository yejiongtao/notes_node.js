1. 相对路径与绝对路径
	html中的都要写成绝对路径！
		包括href、src
	src中 / 指的是你static resources的位置
	href中 / 指的是网站的首页

4. use a column as a link

		th(data-field= col, data-formatter='nameFormatter')=col

		    return '<a href="crash/files/' + value + '">' + value + '</a>';
		}
